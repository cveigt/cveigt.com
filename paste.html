Paste Design System

Building accessible components for Twilio's design system.

Context

Twilio is a customer engagement platform that enables developers to build unique communication experiences. Paste is Twilio's design system, providing the building blocks for creating consistent and accessible products.

My role

As a design systems designer, I worked on creating and maintaining accessible components, collaborating closely with engineers and other designers to ensure high-quality implementation.

Problem

The challenge was to create components that were not only visually consistent but also met WCAG 2.1 accessibility standards while being flexible enough to serve multiple product teams' needs.

Solution

We established a robust component development process that included accessibility reviews, thorough documentation, and extensive testing to ensure our components met both design and technical requirements.

Component design

Each component went through multiple rounds of design reviews, focusing on visual design, interaction states, and accessibility considerations.

Implementation

Working closely with engineers, we implemented components using React and TypeScript, ensuring they were performant and met our accessibility requirements.

Documentation

Comprehensive documentation was crucial for adoption. Each component included usage guidelines, props documentation, accessibility considerations, and examples.

Results

The system was successfully adopted across Twilio's product suite, leading to:

  • Improved accessibility across products
  • Faster implementation time for product teams
  • Consistent user experience
  • Reduced design and development overhead
